# 🎨 Interactive Drawing App

A modern, high-performance web application that allows users to sketch and draw directly in the browser. This project demonstrates the usage of the HTML5 Canvas API, advanced DOM event handling, and dynamic CSS styling featuring a sleek dark mode UI with a clean white canvas.

## 🛠️ Tech Stack & Concepts

* **Languages:** HTML5, CSS3, JavaScript (Vanilla ES6+)
* **HTML5 Canvas API:** Utilizes the `2D Context` for smooth, real-time rendering with rounded brush tips (`lineCap = 'round'`).
* **Event Handling:** Implements efficient control flows using Event Delegation (`click` & `change` listeners attached to the toolbar).
* **Responsive Design:** Automatically calculates canvas dimensions dynamically based on the viewport size minus the toolbar offset.

## 🎛️ App Features
* **Dynamic Brush Size:** Adjust the line width in real time using a range slider.
* **Color Picker:** Choose any color from the native system color palette.
* **Clear Canvas:** Wipe the entire drawing board clean with a single click.
